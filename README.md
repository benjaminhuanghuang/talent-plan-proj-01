## Setup

```
  # code lint
  rustup component add clippy
  cargo clippy

  # code format
  rustup component add rustfmt
  cargo rustfmt


  # create project
  cargo new kvs --lib
```


## Dependencies
- Clap
Clap is a full featured, fast Command Line Argument Parser for Rust.
